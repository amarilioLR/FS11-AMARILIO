Formas de aplicar o CSS

Inline

Descrição:estilo aplicado diretamente no elemento HTML,usando o atributo style.
vantagens:Rápido, útil para testes ou ajustes pontuais.
Desvantagens: Código repetitivo, difícil de manter organizado em projetos de larga escala.

Internal

Descrição:estilo aplicado dentro da tag "style" no próprio arquivo HTML, geralmente no < head> .
vantagens:organiza o estilo em um único lugar dentro da página.
Desvantagens: Não reaproveita estilos para várias páginas.
